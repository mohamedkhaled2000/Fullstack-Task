## Steps to run project

1- clone project

```
git clone <project-url>
```

2- install package

```
composer install
```

3- copy .env.example to create .env file

```
copy .env.example .env
```

4- generate App Key

```
php artisan key:generate
```

5- set database in `.env` file

6- run project serve

```
php artisan serve
```
